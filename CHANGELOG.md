# Changelog

## [v1.0.0] - 2026-04-17

### Features
- React + TypeScript + Vite web app
- Firebase Firestore data storage with localStorage cache
- Google Sign-In authentication
- Sophtron bank integration for checking balance
- Budget tracking with bills, settings, and period overrides
- Export data to Google Drive
- Modern UI with gradient theme, Inter font, and animations
